ngx-timezones

An Angular module for timezones (based on moment.js)

View on GitHub View on NPM Download .zip Download .tar.gz

Selection timezones can be a hurdle when you want to support all timezones for each country in the world. This Angular library utilizes ngx-mat-select-search to make an advanced timezone selector.

Installation

Install the library in your Angular projects root folder:

npm i ngx-timezones
